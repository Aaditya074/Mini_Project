Overview:
This project implements an attendance bot using UiPath. The bot is designed to monitor students' attendance and send email notifications to those with attendance below 75%.

Features:
Monitors students' attendance.
Sends email notifications to students with attendance below 75%.
Automates repetitive tasks related to attendance management.
Requirements

UiPath Studio:
Mail server credentials for sending emails
Excel sheet containing attendance data

Installation:
Clone or download the project to your local machine.
Open the project in UiPath Studio.

Configuration:
Set up your mail server credentials in the bot for sending emails.
Provide the path to the Excel sheet containing attendance data.

Usage:
Run the bot in UiPath Studio.
Monitor the bot's execution to ensure correct functioning.
Check email notifications sent to students with attendance below 75%.
